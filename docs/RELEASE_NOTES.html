Resolved in V0.7.1 — Build 5040
💿

ISO Creator — oscdimg / Windows ADK dependency

Invoke-Oscdimg.ps1 — 3-tier fallback chain
Before

If oscdimg was absent and the ADK silent install failed or needed a restart, the ISO build silently aborted with no output file.

After

Tier 1: oscdimg (ADK auto-install). Tier 2: wimlib-imagex. Tier 3: .NET ZipFile — always available, produces archive with clear warning. No silent failure.

📦

ISO App Embedding — winget not available on target

Install-Apps.bat — self-bootstrapping winget
Before

The generated Install-Apps.bat exited immediately if winget was absent, requiring the user to manually install App Installer first.

After

Script bootstraps winget automatically: Add-AppxPackage -RegisterByFamilyName, then fallback to aka.ms/getwinget. Pre-accepts source agreements silently.

🍫

Chocolatey — separate manual install required

App Manager install job — auto-bootstrap from community.chocolatey.org
Before

When winget failed and an app had a Chocolatey ID, the install was silently skipped if choco.exe was not in PATH.

After

Install job downloads and runs the official Chocolatey bootstrap script, updates PATH, then retries. Full output logged to WinTooler log file.

🚀

Startup Manager — TaskScheduler service dependency

Enable/Disable handlers + schtasks.exe CLI fallback
Before

Enabling or disabling a scheduled-task startup entry silently failed if the Windows Task Scheduler service (Schedule) was stopped.

After

Handlers check Get-Service Schedule, call Start-Service with 800ms settle if stopped, then use schtasks /Change /TN ... /DISABLE|ENABLE as a COM-independent CLI fallback.

🔤

Segoe MDL2 / Fluent icons — Windows 10 < 19041

XAML FontFamily fallback + runtime VisualTreeHelper walk
Before

All sidebar nav icons and tool card icons were invisible on Windows 10 before build 19041 (20H1) since Segoe MDL2 Assets is not fully available.

After

XAML FontFamily extended to Segoe MDL2 Assets, Segoe Fluent Icons, Segoe UI Symbol. If OSBuild < 19041, a VisualTreeHelper walk patches all icon TextBlocks to Segoe UI Symbol at load time.
